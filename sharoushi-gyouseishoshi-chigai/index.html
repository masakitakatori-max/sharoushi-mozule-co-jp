社労士と行政書士の違いは？難易度・仕事内容を比較

社労士（社会保険労務士）と行政書士はどちらも人気の国家資格ですが、仕事内容・難易度・年収は大きく異なります。「どちらを取るべき？」「ダブルライセンスの価値は？」と迷う方も多いでしょう。この記事では社労士と行政書士の違いを7つの観点から徹底比較し、あなたに向いている資格を判断する基準を解説します。

社労士と行政書士の基本比較

| 比較項目 | 社労士 | 行政書士 | |---------|--------|---------| | 正式名称 | 社会保険労務士 | 行政書士 | | 管轄省庁 | 厚生労働省 | 総務省 | | 主な業務 | 人事労務・社会保険手続き | 官公署への許認可申請 | | 独占業務 | 労働・社会保険の書類作成・提出代行 | 許認可申請書類の作成・提出代行 | | 合格率 | 5〜7% | 10〜12% | | 勉強時間 | 800〜1,200時間 | 600〜1,000時間 | | 受験資格 | あり（学歴・実務・他資格） | なし（誰でも受験可） | | 試験時期 | 8月第4日曜日 | 11月第2日曜日 |

仕事内容の違い

社労士の仕事：「ヒト」の専門家 社労士は企業の人事労務管理を支援する専門家です。社労士の仕事内容にあるとおり、3つの独占業務（書類作成・帳簿作成・コンサルティング）を中心に業務を行います。

  • 社会保険・労働保険の手続き代行
  • 就業規則の作成・変更
  • 給与計算・年末調整
  • 労務相談・ハラスメント対策
  • 助成金申請

行政書士の仕事：「許認可」の専門家 行政書士は官公署に提出する書類の作成と許認可申請の代理を行います。詳しくは行政書士とは？仕事内容と役割をご参照ください。

  • 建設業許可・飲食店営業許可等の申請
  • 法人設立手続き
  • 相続手続き・遺言書作成
  • 外国人の在留資格申請
  • 自動車登録手続き

ポイント： 社労士は「企業と従業員の関係」に特化し、行政書士は「市民と行政の橋渡し」に特化しています。

難易度の違い

| 比較項目 | 社労士 | 行政書士 | |---------|--------|---------| | 合格率 | 5〜7% | 10〜12% | | 勉強時間 | 800〜1,200時間 | 600〜1,000時間 | | 試験形式 | 選択式＋択一式 | 択一式＋記述式 | | 科目数 | 10科目 | 6分野 | | 足切り | あり（全科目に基準点） | あり（一般知識に足切り） | | 受験資格 | あり | なし |

社労士の難易度は行政書士よりやや高く、合格率で約5ポイント差があります。ただし試験の性質が異なるため、単純な比較は困難です。社労士は暗記量が膨大で、行政書士は法的思考力が問われる傾向があります。

年収の違い

| 項目 | 社労士 | 行政書士 | |------|--------|---------| | 勤務平均年収 | 450〜600万円 | 400〜500万円 | | 開業平均年収 | 500〜800万円 | 400〜700万円 | | 高年収層（上位10%） | 1,000万円以上 | 1,000万円以上 |

社労士の年収は行政書士より平均で50〜100万円程度高い傾向があります。これは社労士の顧問契約（月額3〜5万円×顧問先数）による安定収入が大きいためです。

どちらの資格が向いている？

社労士が向いている人：

  • 企業の人事・総務で働いている、または経験がある
  • 安定した顧問収入に魅力を感じる
  • 労働法や社会保険に興味がある
  • 大学卒業等の受験資格を満たしている

行政書士が向いている人：

  • 受験資格を気にせず挑戦したい
  • 幅広い業務分野に興味がある
  • 相続・外国人関連の業務に興味がある
  • まず取りやすい資格から始めたい

ダブルライセンスの価値

社労士のダブルライセンスとして行政書士を取得すると、以下のシナジーが生まれます。

  • 会社設立のワンストップ対応： 行政書士で法人設立＋社労士で社会保険手続き
  • 建設業の総合支援： 行政書士で建設業許可申請＋社労士で労働保険手続き
  • 外国人雇用支援： 行政書士で在留資格申請＋社労士で社会保険加入手続き

社労士試験の受験資格には「行政書士合格者」が含まれているため、行政書士→社労士の順で取得するのが効率的なルートです。

まとめ

社労士と行政書士は専門分野が異なる資格です。企業の人事労務に興味があれば社労士、許認可申請や幅広い業務に興味があれば行政書士を選びましょう。社労士の勉強法で学習計画を立てて、資格取得を目指してください。
