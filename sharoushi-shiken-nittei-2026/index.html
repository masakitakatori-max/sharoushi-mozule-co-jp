2026年度（第58回）社労士試験の日程を網羅的にまとめました。結論として、試験日は2026年8月下旬（第4日曜日）が有力で、受験申込は4月中旬〜5月末が目安です。この記事では、申込から合格発表までの全スケジュールと、逆算した学習計画を解説します。

2026年度 社労士試験の主要日程

| イベント | 予定時期 | |---------|---------| | 試験公告 | 2026年3月上旬 | | 受験案内・申込書の配布 | 2026年4月中旬〜 | | 受験申込期間 | 2026年4月中旬〜5月31日（予定） | | 受験票の送付 | 2026年8月上旬 | | 試験日 | 2026年8月23日（日）予定 | | 合格発表 | 2026年10月下旬 |

※2026年度の正式な日程は、2026年3月上旬の官報公告で確定します。上記は過去の傾向に基づく予測です。必ず全国社会保険労務士会連合会の公式サイトで最新情報を確認してください。

試験当日のタイムスケジュール

社労士試験は1日で「選択式」と「択一式」の2つの試験を行います。

| 時間 | 内容 | |------|------| | 10:30〜11:50 | 選択式試験（80分） | | 11:50〜13:20 | 昼休み | | 13:20〜16:50 | 択一式試験（210分） |

選択式80分＋択一式210分の合計290分（約5時間）に及ぶ長丁場の試験です。体力・集中力の配分も重要な要素となります。

受験申込の方法と注意点

申込方法

社労士試験の申込は、以下の2つの方法があります。

1. インターネット申込（推奨）

全国社会保険労務士会連合会の試験センターWebサイトから申し込めます。24時間対応で手続きも簡単です。2026年現在、インターネット申込が主流です。

2. 郵送申込

受験案内を入手し、必要書類を郵送で提出する方法です。受験案内は試験センターのWebサイトからの請求、または都道府県社会保険労務士会の窓口で入手できます。

受験料

2026年度の受験手数料は15,000円（予定）です。払い込み方法は、インターネット申込の場合はクレジットカードまたはコンビニ払い、郵送申込の場合は払込用紙を使用します。

申込時の注意点

  • 受験資格を証明する書類の準備が必要（受験資格の詳細）
  • 申込期間の締切は厳守（郵送の場合は消印有効）
  • 受験地は選択制（全国の主要都市）
  • 写真の準備（パスポートサイズ）

試験会場

社労士試験は全国の主要都市で実施されます。2026年度の試験会場は各都道府県の大学や会議施設が予定されています。

受験申込時に希望する試験地（都道府県）を選択しますが、具体的な試験会場は受験票で通知されます。希望する試験地の近くに会場がない場合は、隣接する都道府県を選択する必要があります。

合格発表と合格後の流れ

合格発表

合格発表は2026年10月下旬に行われます。以下の方法で確認できます。

  • 全国社会保険労務士会連合会のWebサイトに合格者の受験番号を掲載
  • 厚生労働省のWebサイトにも掲載
  • 合格者には合格証書が郵送

合格後の登録

社労士として活動するためには、合格後に全国社会保険労務士会連合会に登録する必要があります。登録には実務経験2年以上、または事務指定講習の修了が必要です。

今から始めて2026年試験に間に合うか

2026年現在、試験まで残り約6か月です。今からのスタートでも合格は可能ですが、以下の条件が重要です。

  • 1日3〜4時間の学習時間を確保できること
  • 通信講座で効率的に学ぶこと
  • 過去問中心のアウトプット重視の学習を行うこと

時間的に厳しい場合は、2027年試験を目標にじっくり準備するのも賢い選択です。

よくある質問（FAQ）

Q. 試験日が変更になることはありますか？

過去に台風などの自然災害で一部地域の試験が延期されたケースがあります。正式な日程は官報で公告されますので、必ず確認しましょう。

Q. 受験地は自由に選べますか？

はい、全国の試験地から自由に選択できます。自宅や職場に近い試験地を選ぶのが一般的です。

Q. 受験申込を忘れた場合、追加申込はできますか？

原則として、申込期間を過ぎると受験できません。申込期間の開始と同時に手続きすることをおすすめします。

まとめ

2026年度の社労士試験は8月23日（日）が有力です。受験申込は4月中旬から始まるため、必要書類の準備は早めに進めておきましょう。

試験まで逆算した学習計画を立て、計画的に準備を進めることが合格への鍵です。効率的な学習には通信講座の活用がおすすめです。
