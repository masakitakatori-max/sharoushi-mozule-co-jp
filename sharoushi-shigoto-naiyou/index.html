社労士の仕事内容を3つの業務で解説

社労士（社会保険労務士）は具体的にどんな仕事をするのか。結論として、1号業務（手続代行）・2号業務（帳簿作成）・3号業務（コンサルティング）の3つが社労士の主な業務です。この記事では、各業務の詳細と勤務社労士・開業社労士の違いを解説します。

社労士の3つの業務

社労士の業務は社会保険労務士法で定められており、大きく3つに分類されます。

| 業務区分 | 内容 | 独占業務 | |---------|------|---------| | 1号業務 | 労働社会保険の手続代行 | ○（社労士の独占） | | 2号業務 | 帳簿書類の作成 | ○（社労士の独占） | | 3号業務 | コンサルティング・相談業務 | ×（誰でも可能） |

1号業務と2号業務は社労士の独占業務であり、資格を持っていない人が報酬を得て行うことはできません。この独占業務があることが、社労士資格の大きな価値です。

1号業務：労働社会保険の手続代行

1号業務は、企業に代わって行政機関への各種届出・申請を行う業務です。

具体的な業務内容

  • 雇用保険：入社・退社時の資格取得届・喪失届の提出
  • 社会保険：健康保険・厚生年金の資格取得届・喪失届の提出
  • 労働保険：労働保険の年度更新手続き
  • 労災保険：業務災害・通勤災害の給付申請
  • 助成金：各種助成金の申請代行

特に中小企業では、社会保険や労働保険の手続きに詳しい担当者がいないケースが多く、社労士が代行することで企業の負担を軽減します。

1号業務の重要性

2026年現在、社会保険や雇用保険の手続きは電子申請が主流になりつつありますが、手続きの複雑さは変わりません。法改正に対応した正確な手続きを行うためには、専門家である社労士の存在が不可欠です。

2号業務：帳簿書類の作成

2号業務は、労働社会保険に関する帳簿書類を作成する業務です。

具体的な業務内容

  • 就業規則の作成・変更
  • 賃金台帳の作成
  • 労働者名簿の作成
  • 出勤簿の整備
  • 36協定（時間外労働・休日労働に関する協定）の作成

特に就業規則の作成・変更は、社労士の重要な業務の一つです。労働トラブルを未然に防ぐためには、法律に適合した就業規則が不可欠であり、企業からの需要が高い業務です。

3号業務：コンサルティング

3号業務は、労務管理や社会保険に関する相談・指導を行う業務です。独占業務ではありませんが、社労士の専門性が最も活きる分野です。

具体的な業務内容

  • 人事制度の設計・導入支援
  • 賃金制度の見直し
  • 労働トラブル（解雇・ハラスメント等）の相談対応
  • メンタルヘルス対策の提案
  • 働き方改革への対応支援
  • 年金相談

2026年現在、働き方改革やハラスメント防止法の施行に伴い、3号業務の需要が急速に拡大しています。AIでは代替しにくい「人」に関わる相談業務は、社労士の将来性を支える柱となっています。

勤務社労士と開業社労士の違い

社労士の働き方は大きく「勤務」と「開業」に分かれます。

| 比較項目 | 勤務社労士 | 開業社労士 | |---------|-----------|-----------| | 勤務形態 | 企業に雇用される | 独立して事務所を開く | | 業務範囲 | 自社の労務管理が中心 | 複数のクライアント企業を担当 | | 収入 | 給与（安定的） | 報酬（実力次第で高収入可能） | | リスク | 低い | 顧客獲得が必要 | | 向いている人 | 安定志向、企業内キャリア | 独立志向、営業力がある人 |

勤務社労士の1日

| 時間 | 業務内容 | |------|---------| | 9:00 | 出社。社会保険の届出書類チェック | | 10:00 | 従業員からの年金・保険相談対応 | | 11:00 | 就業規則の改定作業 | | 13:00 | 人事部門のミーティング参加 | | 14:00 | 労働保険の年度更新手続き | | 16:00 | 労働基準監督署からの調査対応 | | 17:30 | 退社 |

開業社労士の1日

| 時間 | 業務内容 | |------|---------| | 9:00 | メール確認、顧問先からの問い合わせ対応 | | 10:00 | 顧問先A社へ訪問。労務相談対応 | | 12:00 | 昼食を兼ねた異業種交流会 | | 13:30 | 事務所で助成金申請書類の作成 | | 15:00 | 新規顧問先の営業活動 | | 16:30 | 就業規則の改定案作成 | | 18:00 | 業務終了 |

社労士が活躍する場所

社労士は多様なフィールドで活躍しています。

  • 企業の人事・総務部門（勤務社労士）
  • 社労士事務所（開業社労士または社員として）
  • コンサルティングファーム
  • 行政機関（年金事務所、労働基準監督署の相談員）
  • 教育機関（社労士講座の講師）

よくある質問（FAQ）

Q. 社労士は事務作業だけですか？

いいえ、3号業務（コンサルティング）は企業の経営者や従業員と対面で相談する業務であり、コミュニケーション能力が求められます。事務作業だけではありません。

Q. AIに仕事を奪われませんか？

1号・2号業務の一部はAIで効率化される可能性がありますが、3号業務（コンサルティング）はAIでは代替困難です。詳しくは「社労士の年収は？勤務・開業別に徹底解説」をご覧ください。

Q. 未経験でも社労士の仕事はできますか？

試験に合格し、登録すれば業務を行えます。ただし、実務経験がない場合は事務指定講習の受講が必要です。

まとめ

社労士の仕事内容は手続代行・帳簿作成・コンサルティングの3つの業務で構成されています。特に3号業務のコンサルティングは、働き方改革の推進に伴い需要が拡大しており、社労士の将来性は明るいと言えます。
