LEC東京リーガルマインドの社労士講座は、法律系資格に強い老舗予備校として高い実績を持っています。結論として、講師の質とカリキュラムの充実度に定評がある一方、受講料が高めで通信特化型講座と比べるとコスパに課題がある講座です。この記事では、LEC社労士講座の特徴・料金・口コミを詳しく解説します。

LEC社労士講座の基本情報

| 項目 | 内容 | |------|------| | 講座名 | LEC東京リーガルマインド 社会保険労務士講座 | | 受講料 | 約150,000〜245,000円（コースにより異なる） | | 学習形態 | 通学（生講義・ビデオ）＋通信（Web・DVD） | | テキスト | オリジナルテキスト | | 講義時間 | 約180時間以上 | | 質問対応 | あり（教えてチューター・教室質問） | | 合格率 | 非公開（合格者の声は掲載） | | 教育訓練給付金 | 対象コースあり |

LEC東京リーガルマインドは1979年創業の老舗資格予備校です。2026年現在、司法試験・行政書士・社労士など法律系資格を中心に講座を展開しており、法律教育のノウハウが豊富です。

LEC社労士講座の5つの特徴

特徴1：法律教育に強い老舗の信頼感

LECは法律系資格の指導実績が40年以上あります。社労士試験も労働法・社会保険法が出題の中心であり、LECの法律教育のノウハウが活きる分野です。テキストの法律解説は正確で網羅性が高いと評価されています。

特徴2：実力派講師陣の充実

LECの社労士講座は複数の人気講師から選べる点が特徴です。それぞれの講師が独自のアプローチで授業を展開しており、無料体験で自分に合った講師を見つけることができます。講師によって教え方のスタイルが異なるため、相性の良い講師を選ぶことで学習効率が大きく変わります。

特徴3：通学と通信の選択肢

TACと同様に通学講座を提供しています。全国の主要都市に校舎があり、生講義を受けられる環境が整っています。もちろん通信講座（Web通信・DVD通信）も選択可能で、通学と通信を併用する受講スタイルも取れます。

特徴4：模試の質が高い

LECの社労士模試は受験者数が多く、自分の実力を正確に測れると好評です。本試験の出題傾向を踏まえた予想問題は的中率が高いと評判で、模試のみの受験も可能です。

特徴5：再受講割引制度

過去にLECの社労士講座を受講した方は、翌年以降の再受講時に大幅な割引が適用されます。不合格でも翌年のコストを抑えられる点は、長期的に見るとメリットです。

LEC社労士講座の良い口コミ・評判

「講師の説明が丁寧でわかりやすい」

「法律の趣旨から丁寧に解説してくれるので、暗記ではなく理解できた」という声が目立ちます。LECの講師は法律系資格の指導経験が豊富で、条文の背景や実務との関連を交えた説明が高く評価されています。

「テキストの内容が充実している」

「網羅性が高く、本試験で知らない論点がほとんどなかった」という口コミがあります。LECのテキストは情報量が多く、しっかり読み込めば十分な知識が身につきます。

「模試の予想が当たった」

「本試験で似たような問題が出て助かった」という声もあります。LECの全国模試は社労士受験生の間で信頼度が高く、直前期の実力チェックに最適です。

LEC社労士講座の悪い口コミ・評判

「受講料が高い」

LECの受講料は15万〜25万円と、通信特化型の講座と比べて割高です。特に通信講座のみの利用では「コスパが良くない」と感じる方もいます。

「テキストが分厚くて読みきれない」

網羅性の裏返しとして、テキストの分量が多い点はデメリットにもなります。「初学者には情報量が多すぎて、何が重要かわかりにくい」という声もあります。

「デジタル対応が他社に劣る」

スマホ学習の利便性では、スタディングやフォーサイトに比べて見劣りする部分があります。スキマ時間にスマホで学習したい方には物足りなさがあるかもしれません。

LEC社労士講座が向いている人・向いていない人

向いている人

  • 法律の趣旨からしっかり理解したい人
  • 講師を選んで学びたい人
  • 通学で学びたい人
  • 網羅的なテキストで学びたい人
  • 模試の質を重視する人

向いていない人

  • 受講料を抑えたい人
  • コンパクトな教材で効率的に学びたい人
  • スマホ中心のスキマ学習をしたい人
  • 合格実績を数値で確認したい人

よくある質問（FAQ）

Q. LECとTACならどちらがおすすめですか？

どちらも大手予備校で質が高いですが、講師を選びたいならLEC、答練・自習室を重視するならTACがおすすめです。無料体験で比較することをおすすめします。

Q. LECの通信講座だけで合格できますか？

はい、通信講座のカリキュラムは通学と同じ内容です。講義動画・テキスト・模試をしっかり活用すれば、通信だけでも十分合格を目指せます。

Q. LECの模試だけ受けることはできますか？

はい、全国公開模試は外部生も受験可能です。他社の講座を受講しながらLECの模試を活用する受験生も多くいます。

まとめ

LEC社労士講座は老舗予備校ならではの講師の質とカリキュラムの充実度が魅力です。法律の趣旨から丁寧に学びたい方には最適な環境が整っています。

一方で、受講料の高さやデジタル対応の弱さは検討ポイントです。通信講座のみで受講を考えている方は、コスパの面で他社講座も比較してみてください。
